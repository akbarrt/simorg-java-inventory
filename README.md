# SIMORG  
**Smart Inventory Management for Organization**

> Kelola aset organisasi dengan rapi, cepat, dan terkontrol.

SIMORG adalah aplikasi desktop berbasis Java yang dirancang untuk membantu organisasi dalam mengelola data inventaris secara terstruktur. Aplikasi ini mendukung pencatatan barang, pengelolaan stok, serta proses peminjaman dan pengembalian barang dengan pendekatan Object-Oriented Programming (OOP).
